=== Description === 
This software will list the users who have logged in within the last hour, allowing you to keep track of who is logging in on the dashboard.
There is no warranty provided with this software.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/your-plugin` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
